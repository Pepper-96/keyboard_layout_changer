# Keyboard Layout Changer

Небольшая фоновая утилита для Windows, которая исправляет текст, набранный не в той раскладке.  
Классический пример: вместо `Привет` введено `Ghbdtn` — программа по горячей клавише автоматически заменяет выделенный текст на корректный.

## Возможности

- Работает в фоновом режиме, без окон (иконка в системном трее).
- Глобальная горячая клавиша для исправления раскладки (по умолчанию `Ctrl+Shift+Q`, можно поменять в коде).
- Автоматически определяет направление преобразования по самому тексту:
  - `Ghbdtn` → `Привет` (EN → RU)
  - `Руддщ` → `Hello` (RU → EN)
- Аккуратно работает с буфером обмена: сохраняет старое содержимое и восстанавливает его после операции.
- Одноразовый запуск: защита от нескольких одновременно запущенных экземпляров.

## Технологии

- Python 3
- `pywin32` (работа с буфером обмена, WinAPI)
- `keyboard` / `pynput` (глобальные хоткеи и эмуляция нажатий)
- `pystray` + `Pillow` (иконка в системном трее)
- `PyInstaller` для сборки в один exe

## Как это работает

1. Пользователь выделяет «криво» набранный текст в любом приложении.
2. Нажимает горячую клавишу.
3. Программа:
   - копирует выделенный текст в буфер обмена,
   - определяет, какая раскладка «ожидалась», по набору символов,
   - конвертирует символы по таблице соответствий,
   - подставляет исправленный текст обратно в поле ввода.
